# Stock Average Calculator

## Project Description
The Stock Average Calculator is a simple tool designed to help users calculate the average price of their stock purchases. By inputting their stock transactions, users can easily determine their average cost per share, aiding in better financial decision-making.

## Features
- Calculate average stock price based on multiple transactions.
- User-friendly interface for inputting stock data.
- Support for various types of stock transactions (buy/sell).
- Provides the ability to export calculations for record-keeping.

## Installation Instructions
1. Clone the repository:
   ```bash
   git clone https://github.com/VishnuMygithub/StockAverageCalculator.git
   ```
2. Navigate to the project directory:
   ```bash
   cd StockAverageCalculator
   ```
3. Install necessary dependencies:
   ```bash
   npm install
   ```

## Usage Guide
1. Open the application:
   ```bash
   npm start
   ```
2. Enter your stock transaction details:
   - For each purchase, input the number of shares and the price per share.
   - For sales, input the number of shares sold.
3. View the calculated average stock price.
4. (Optional) Export your transaction data.
